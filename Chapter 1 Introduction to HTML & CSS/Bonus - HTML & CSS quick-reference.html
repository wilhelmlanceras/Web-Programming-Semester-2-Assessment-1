HTML Tags

1. <title>

This is the title that appears on the tab.

<title>Page Title</title>

2. <meta>

Provides metadata about the web page, such as charset.

<meta charset="UTF-8">

3. <h1> to <h6>

Defines HTML headers of varying sizes.

<h1>Heading Level 1</h1>

4. <p>

This defines a paragraph of text.

<p>This is a paragraph.</p>

5. <br>

It inserts a line break.

<br>

6. <dl>

Defines a description list. Can contain <dt> and <dd> elements.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
</dl>

7. <ul> and <ol>

This defines unordered and ordered lists. List items are created with <li>.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

8. <code> and <pre>

This is used to display code examples.

<pre>
<code>code sample</code>
</pre>

9. <img>

This is used to display images on the page.

<img src="image.jpg" alt="Description">

10. <div>

This defines a division or section in an HTML document. It is often used to group elements together.

<div>
  <h2>Section Heading</h2>
  <p>This is content inside a div element.</p>
</div>

Common CSS Properties

1. background-color

Sets the background color of an element.

div {
  background-color: #f4f4f4;
}

2. color

Sets the text color of an element.

p {
  color: blue;
}

3. font-family

Sets the font of an element.

body {
  font-family: Arial, sans-serif;
}

4. font-size

Sets the size of the font.

h1 {
  font-size: 2em;
}

5. font-style

Sets the font style (example, italic).

p {
  font-style: italic;
}

6. font-weight

Sets the thickness of the font (example, bold).

h2 {
  font-weight: bold;
}

7. line-height

Sets the amount of space between lines of text.

p {
  line-height: 1.6;
}

8. margin

Sets the space around an element, outside its border.

div {
  margin: 20px;
}

9. text-align

Sets the horizontal alignment of text inside an element.

h1 {
  text-align: center;
}

10. text-decoration

Sets the decoration for text (example, underline).

a {
  text-decoration: none;
}
